# KNOWLEDGE — Curated Learnings

Inhalte werden täglich vom Daily-Sync (20:00 Berlin) aus allen Source-Repos aggregiert. Dedup + Kategorisierung passiert automatisch.

| Datei | Inhalt |
|---|---|
| `WHAT_WORKED.md` | Patterns die in mind. 1 Projekt funktioniert haben |
| `WHAT_FAILED.md` | Anti-Patterns / Was-NICHT-tun |
| `GOTCHAS.md` | Cross-Stack Stolpersteine (Windows, OOM, ESM, …) |
| `METHODOLOGY.md` | Workflow-Patterns (PRD, Loop, Run-Log, DoD) |
| `AGENT_LEARNINGS/` | Pro Agent: Stärken / Bias / Bewährte Prompts |
| `TOOLS/` | Tool-Recipes (Ralph, Caveman, MCP-Connectors) |

**Eintrags-Format:** Jede neue Erkenntnis bekommt eine ID (G01, G02, ... pro Kategorie), "Erstmals beobachtet"-Datum, "Beobachtet in"-Liste, "Was passiert", "Fix", "Quellen".
